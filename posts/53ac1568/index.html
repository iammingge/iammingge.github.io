stm32CubeMX

安装

下载链接 stm32CubeMX

注意 ！！！ ： 需要 java 运行环境，下载链接 jre—java运行环境

搭建基本工程

选择芯片并新建工程

  • 点击 ACCESS TO MCU SELECTOR

  • 选择型号，然后点击 Start Project

配置外设参数

  • 调试接口

  • 其他外设参数配置见 后续文章

配置时钟

  • 先配置主时钟（HCLK）,之后根据需要配置外设时钟。

生成编译环境对应的工程

  • Project

    Project Name：项目名

    Project Location：项目工作区

    Application Structure：项目结构

    Basic：基础结构

    Advanced：复杂结构，一般使用中间件时选此选项

    ▨ Do not generate the main()：不生成 main() 函数，不勾选自动生成 main() 函数

    Toolchain Folder Location：工程文件夹

    Toolchain / IDE：编译环境

    ▨ Generate Under Root：在根目录下生成工程，不勾选可以不在根目录下生成工程，修改工程文件夹路径，目前，只有 SW4STM32、TrueSTUDIO、STM32CubeIDE 可以不在根目录下生成工程，其他编译环境该项不可以选，默认根目录下生成工程

    Linker Settings：堆栈设置

    Minimum Heap Size：最大堆空间，如果使用的 heap 超过设置值会运行错误

    Minimum Stack Size：最大栈空间，可以随意设置因为芯片会自动分配

    Mcu Reference：芯片型号

    Firmware Package Name and Version：使用的固件包版本

    ▨ Use lastest available version：使用最新固件包，不勾选可以选择老版本固件包

    ▨ Use Default Firmware Location：使用默认固件包位置，不勾选需要自己导入固件包路径

  • Code Generator

    ◉ Copy all used libraries into the project folder：复制所有库源文件到工程中

    ◉ Copy only the necessary library files：只复制需要的库源文件到工程中

    ◉ Add necessary library files as reference in the toolchain project configuration file：不复制库源文件，调用库源文件，工程中没有库源文件

    ▨ Generate peripheral initialization as a pair of’.c/.h’ files per peripheral：初始化代码生成在对应的外设文件，不勾选所有初始化代码都生成在 main.c

    ▨ Backup previously generated files when re-generating：在重新生成时，备份以前生成的文件到 Backup 文件夹，不勾选不备份

    ▨ Keep User Code when re-generating：在重新生成时，保留用户代码，不勾选不保留

    ▨ Delete previously generated files when not re-generated：在重新生成时，删除无用的文件，不勾选不删除

    ▨ Set all free pins as analog (to optimize the power consumption)：设置所有空闲引脚为模拟模式（避免灌电流，以优化功耗），不勾选不设置

    ▨ Enable Full Assert：开启断言，不勾选不开启

    Select a template to generate customized code：选择一个模板生成定制代码（一般用不到）

  • Advanced Settings

    Not Generate Function Call：工程生成后不调用初始化函数，不勾选工程自动调用（经常用在初始化外设有先后顺序的情况，需要根据需要自行调用）

    Visibility (Static)：初始化函数为 static 型，不勾选不限制（此选项有在 Generate peripheral initialization as a pair of’.c/.h’ files per peripheral 不勾选的前提下奏效）

​ 最后点击右上按钮 GENERATE CODE：生成代码，一个完整的工程就生成好了 ！！！
